# Money 주식 매매타점 분석 엔진

이 프로젝트는 주식 가격을 예측하는 도구가 아니라, 국내/미국 주식의 공개 데이터를 수집하고 차트와 보조지표를 활용해 조건부 매매 판단을 돕는 실전 리포트 생성기다.

목표는 투자 수익을 보장하는 것이 아니라, 사용자가 지금 해야 할 행동과 피해야 할 행동을 구분하도록 돕는 것이다.

보고서는 짧고 행동 중심으로 작성된다.

일반 종목 분석 요청 시에는 해당 종목만 분석하며, 회귀테스트는 자동으로 실행하지 않는다.

분석 결과는 매수 강요가 아니라 조건부 시나리오다.

## 주요 기능

- 국내 주식과 미국 주식 분석
- 정규장 중 장중 현재가 우선 반영
- 장외에는 최신 완료 일봉 기준 분석
- 가격/거래량 데이터 교차검증
- 이동평균선, 일목균형표, MACD, RSI, 볼린저밴드, ATR, 매물대 계산
- 현재가/매수 관심가/단기 재돌파선/돌파 확인가의 가격 관계 판단
- 프로 트레이더 관점의 오늘 할 행동과 하지 말아야 할 행동 제시
- 짧은 실전 매매 판단 리포트 생성
- 단일 보고서 내부 검증 통과 시에만 정상 보고서 저장

## 기본 실행 방법

프로그램처럼 켜두고 자연어로 분석하려면 아래 명령을 실행한다.

```bash
python money_assistant.py
```

Windows에서는 아래 파일을 더블클릭해도 같은 프롬프트가 열린다.

```text
Money_Assistant.bat
```

`Money_Assistant.bat`은 `KIWOOM_BRIDGE_URL`이 없으면 기본값 `http://127.0.0.1:8765`를 자동 설정하고, 실행 전에 로컬 키움 브릿지 포트 연결 여부를 확인한다. 브릿지가 꺼져 있으면 `Start_Kiwoom_Bridge.bat`을 새 창으로 실행해 `kiwoom_bridge_server/` 안의 32비트 Python 기반 키움 OpenAPI+ 브릿지와 로그인 창이 뜰 수 있게 시도한다.

실행 후 프롬프트에 아래처럼 입력한다.

```text
삼성전자 분석해줘
005930 삼성전자 장중 분석해줘
삼성전자 일봉 분석해줘
삼성전자 키움 조건부 분석해줘
NVDA 분석해줘
```

`삼성전자 분석해줘`처럼 모드를 지정하지 않으면 단계별 게이트형 통합 분석을 실행한다. 공개 데이터 일봉/스윙 분석을 기본 레이어로 사용하고, 로컬 키움 브릿지가 연결되어 있으면 현재가·3분봉·5분봉·SSE Indicator를 안전 필터로 추가 반영한다.

키움 실시간 데이터가 없거나 검증에 실패하면 공개 데이터 기준 큰 그림 분석은 저장하되, 장중 신규매수 긍정 지시는 제한한다.

키움 조건부 명령형 분석은 로컬 키움 브릿지를 먼저 실행해야 한다. 기본 브릿지 주소가 아닌 경우에는 `KIWOOM_BRIDGE_URL`을 직접 설정한다.

장중/장외를 자동 분기해 분석한다.

```bash
python analyze_stock_intraday.py 033100 제룡전기
```

완료 일봉 중심 분석을 실행한다.

```bash
python analyze_stock.py 033100 제룡전기
```

## 회귀 테스트 실행 방법

회귀 테스트는 필수 실행 흐름이 아니라 선택 기능이다.

```bash
python run_regression_tests.py
```

테스트 대상만 확인한다.

```bash
python run_regression_tests.py --dry-run
```

## 보고서 저장 위치

보고서는 종목명과 종목코드가 포함된 폴더에 저장된다.

```text
reports/종목명_종목코드/
```

예:

```text
reports/제룡전기_033100/
```

정상 생성 파일:

```text
종목명_종목코드_매매타점_분석보고서.md
종목명_종목코드_매매타점_분석보고서.html
종목명_종목코드_데이터검증.csv
종목명_종목코드_지표요약.csv
```

## QA 실패 확인 방법

QA 실패 시 정상 Markdown/HTML 보고서는 저장하지 않는다.

대신 아래 파일을 확인한다.

```text
reports/종목명_종목코드/종목명_종목코드_보고서_QA실패.md
```

회귀 테스트 요약은 아래 위치에 저장된다.

```text
test_outputs/regression_summary.md
```

## 주의사항

- 이 프로젝트는 투자 수익을 보장하지 않는다.
- 모든 판단은 조건부 매매 계획이다.
- 장중 데이터는 확정 일봉이 아니므로 종가 확인이 필요하다.
- 데이터 수집 실패 또는 불일치 시 정밀 판단은 제한된다.
- 실제 매매 판단과 책임은 사용자에게 있다.
