# 08_KIWOOM_COMMAND_ANALYZER_RULES - 키움 조건부 명령형 분석 규칙

## 목적

키움증권 API는 자동매매가 아니라 분석 데이터 수집 Provider로만 사용한다.

주문 전송, 시장가 주문, 조건주문, 자동매매 함수는 구현하지 않는다.

## 구조

- `kiwoom/client.py`: 로컬 키움 브릿지/API 연결 담당
- `kiwoom/provider.py`: 현재가, 체결, 일봉, 분봉 데이터 제공
- `kiwoom/candles.py`: 체결 데이터에서 1분봉/3분봉/5분봉 생성
- `kiwoom/models.py`: `Quote`, `Tick`, `Candle` 데이터 모델
- `core/decision_engine.py`: 조건부 명령형 최종 판정 생성
- `command_chart_analyzer.py`: 조건부 명령형 출력 전용 엔트리포인트

## 표준 OHLCV

키움 Provider가 반환하는 차트 데이터는 아래 컬럼을 가진다.

```text
DateTime 또는 Date
Open
High
Low
Close
Volume
TradeValue
```

지표 계산은 표준 OHLCV 변환 이후 수행한다.

## 필수 지표

일봉 필수 지표:

- MA5, MA10, MA20, MA60, MA120, MA240
- BB상단, BB중심, BB하단
- 전환선, 기준선, 선행스팬1, 선행스팬2
- 거래량20평균, 거래량비율20
- ATR14

분봉 필수 지표:

- MA5, MA10, MA20
- BB상단, BB중심, BB하단
- 거래량20평균, 거래량비율20

RSI14와 MACD는 보조 판단으로 유지한다.

## 최종 판정

`core/decision_engine.py`는 아래 판정 중 하나만 반환한다.

- 사라
- 조건부로 사라
- 사지 마라
- 기다려라
- 팔아라
- 보유하라
- 분석 중단

각 판정은 숫자 가격 조건을 포함해야 한다.

## 가격 근거

보고서의 핵심 지지선, 매수 확인선, 돌파선, 추격 금지선, 손절/방어선은 반드시 가격 근거를 가진다.

허용 근거:

- 20일 이동평균선
- 60일 이동평균선
- 볼린저밴드 중심선
- 볼린저밴드 상단/하단
- 일목 전환선
- 일목 기준선
- 일목 구름 상단/하단
- 최근 5일/20일 고점
- 최근 5일/20일 저점
- 3분봉/5분봉 20이평선
- 3분봉/5분봉 볼린저밴드 중심선
- 직전 분봉 반등 고점/저점

근거 없는 가격은 출력하지 않는다.

## 장중/장외 표현

장중에는 `일봉 돌파 확정`이라고 쓰지 않는다.

장중 돌파는 `장중 돌파 시도` 또는 `분봉 기준 조건부 매수`로만 표현하고 3분봉 또는 5분봉 종가 유지 조건을 붙인다.

장마감 이후 확정 일봉에서만 `일봉 종가 돌파 확인`이라고 표현한다.

## QA 실패 저장

아래 상황에서는 정상 `.md/.html` 리포트를 저장하지 않고 QA 실패 파일만 저장한다.

- 키움 현재가 수집 실패
- 키움 체결 데이터 부족
- 분봉 생성 실패
- 대표가격 산정 불가
- 키움 현재가와 일봉 대표 가격의 비정상 불일치
- 필수 지표 계산 실패
- 데이터 신뢰도 낮음
- 장중/장외 상태 판정 실패

이 경우 최종판정은 반드시 `분석 중단`이다.
