# Kiwoom Bridge Server

This folder contains the local Kiwoom OpenAPI+ bridge used by `Money_Assistant.bat`.

It is an analysis-data bridge only. Do not add order, market-order, condition-order,
or automated trading functions here.

Startup:

```bat
..\Start_Kiwoom_Bridge.bat
```

Default endpoint:

```text
http://127.0.0.1:8765
```
